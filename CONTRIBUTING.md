# Contributing

Thanks for taking the time to contribute!

## Getting started

- Fork the repository and create your branch from `main`.
- Install dependencies with `composer install`.
- Run the test suite with `composer test`.

## Coding standards

- Run `composer cs-check` to check coding standards.
- Run `composer cs-fix` to auto-fix when possible.
- Run `composer stan` to run PHPStan.

## Pull requests

- Keep changes focused and include tests where appropriate.
- Update documentation or config samples if behavior changes.
- Make sure CI passes.

By participating, you agree to abide by the Code of Conduct.
